# box-apex-sdk

Box Apex SDK
============

The Box Apex SDK for interacting with the 
[Box Content API](https://box-content.readme.io/).

Quickstart
----------

The SDK can be obtained by cloning the source locally and then using either your IDE or deployment tool of choice.
